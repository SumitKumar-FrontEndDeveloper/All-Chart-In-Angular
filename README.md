# Introduction

The project will show you how to implement chart in the angular 8.

PIE CHART ,  BUBBLE CHART , LINE CHART , DOUGHHUNT CHART , BAR , RADAR CHART

# Instalation

1. Download project 
2. run npm install
3. ng serve
